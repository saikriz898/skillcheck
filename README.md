# SkillCheck

SkillCheck is a backend-first skill evaluation platform built as a course project.

## Tech Stack (Planned)
- Node.js
- Express.js
- PostgreSQL

## Project Status
Phase 0 (Foundation) completed.
Backend development will proceed phase-by-phase with strict discipline.
